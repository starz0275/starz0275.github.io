```
git add .
```

### 2. 保存存档
```
git commit -m "更改的内容"
```

### 3. 部署到 GitHub Pages
```
cmd /C 'set "GIT_USER=starz0275" && yarn deploy'
```

## 重新在新电脑上部署网站

### 第一步：打开 PowerShell 终端
输入以下命令安装 Yarn，Docusaurus 依赖

```PowerShell
npm install -g yarn
```

### 第二步：git clone 自己的仓库

在终端里，进入你想存放项目的目录（比如 D 盘）：

我这放的是F盘下：

```PowerShell
git clone https://github.com/starz0275/starz0275.github.io.git
```
进入项目文件夹：

### 第三步：安装依赖包

在项目文件夹下运行：

```PowerShell
yarn install
```
直到出现 success Saved lockfile 之类的提示。这一步会把 Docusaurus 的核心文件下载到这台电脑上。

### 第四步：测试运行

依赖装好后，立刻测试一下能否启动：

```PowerShell
yarn start
```
浏览器自动弹出了你的网站，环境配置成功了！
