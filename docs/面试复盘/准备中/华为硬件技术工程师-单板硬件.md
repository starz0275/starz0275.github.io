## 一、仪器仪表（必考 2 题｜数字万用表）

1. **数字万用表核心考点**

- 测量档位：电压（交 / 直流）、电流、电阻、二极管、通断、电容档区分
- 测量规则：**电压并联、电流串联**，严禁电流档直接并联电源（烧表）
- 二极管档：测量正向压降，硅管 0.6~0.7V，发光二极管 LED 压降单独记忆
- 精度：直流测量精度＞交流，电阻测量需断电、放电后操作

## 二、信号与系统（超高频｜奈奎斯特 + 傅里叶 各 2 题）

### 1. 奈奎斯特采样定理

- 核心公式：fs​≥2fmax​
- 关键结论：
    
    1. 采样频率必须**大于等于信号最高频率的 2 倍**
    2. 奈奎斯特频率 = 采样频率的一半
    3. 不满足会产生**混叠失真**，无法无失真恢复原始信号
    

### 2. 傅里叶基础

- 核心：任意周期信号 / 非周期信号，可分解为**不同频率正弦波叠加**
- 时域↔频域转换：时域周期→频域离散；时域有限→频域无限
- 硬件场景：电源纹波、EMI 噪声、信号谐波分析常用

## 三、滤波电路｜高低频 + 有源无源

### 1. 高频 & 低频电路区别

表格

|维度|低频电路|高频电路|
|---|---|---|
|分布参数|忽略（只看电阻电容电感）|必须考虑寄生电容、走线电感、趋肤效应|
|干扰|干扰小|串扰、辐射、反射、阻抗匹配问题严重|
|分析方式|基尔霍夫定律|传输线、电磁场理论|

### 2. 有源滤波 VS 无源滤波

- **无源滤波**：纯 R/L/C 搭建，无运放 / 芯片供电
    
    优点：结构简单、成本低、耐高压、无功耗
    
    缺点：带负载能力差、低频滤波体积大、增益不可调
- **有源滤波**：运放 + R/C 组成，需要供电
    
    优点：体积小、滤波效果好、可放大 / 缓冲信号、带载强
    
    缺点：有电源限制、耐压低、高频性能受限

## 四、基础器件：发光二极管 LED

- 正向导通压降：**普通红光 1.8~2.2V**，蓝 / 白光 2.8~3.2V
- 工作方式：单向导通，必须串联限流电阻，防止过流烧毁
- 反向耐压低，反向加电不发光、易击穿

## 五、电磁波 & 波导：TEM 波

1. **TEM 波**：横电磁波

- 电场、磁场**均垂直于传播方向**
- 产生 / 传输：**同轴电缆、双绞线、微带线** 为主

2. 易混区分：

- TE 波：电场横向、磁场有纵向分量
- TM 波：磁场横向、电场有纵向分量
- 硬件单板走线、高速信号传输常考 TEM 波

## 六、数字电路：组合逻辑电路

1. 核心定义：**无触发器、无记忆功能**
2. 特点：输出仅由**当前输入**决定，和历史状态无关
3. 常见器件：与 / 或 / 非门、译码器、加法器、数据选择器
4. 对比时序逻辑：有触发器、有记忆（寄存器、计数器）

## 七、模拟电路重点

### 1. 运算放大器 原理

- 两大理想特性：**虚短、虚断**
    
    虚短：V+​=V−​；虚断：输入端电流 = 0
- 常用电路：比例放大、电压跟随器、积分 / 微分电路、比较器
- 硬件应用：信号调理、滤波、缓冲、电平转换

### 2. 戴维南等效电路（计算题高频）

- 两步解题（笔试必考计算）：
    
    1. 求开路电压 Uoc：负载断开，计算端口两端电压
    2. 求等效内阻 Req：独立源置零（电压源短路、电流源开路），算端口等效电阻
    
- 作用：复杂电路简化，快速分析负载电流、电压

## 一、PCB 设计 & 布局布线（高频必考）

### 1. 布局核心原则

1. 强弱电分区、**数字地 / 模拟地分区**，单点接地，避免地环路干扰
2. 核心芯片、时钟源、高速器件优先紧凑布局，缩短关键走线
3. 电源类器件靠近供电入口，滤波电容**就近芯片电源引脚**
4. 敏感电路（模拟、采样、复位）远离大功率、高频开关器件

### 2. 布线关键规则

- 高速信号线：**阻抗匹配、等长走线、减少过孔**
- 走线避免 90° 直角，优先 45°/ 圆弧，降低反射与 EMI
- 电源线、地线尽量加宽，减小线阻、压降与地弹噪声
- 差分线：等长、等距、平行、同层，严格控制间距

### 3. 层叠设计常识

- 典型四层板：信号层→地层→电源层→信号层
- 完整参考平面是**阻抗稳定、抑制串扰**的核心
- 电源 / 地平面紧邻，增大寄生电容，降低电源纹波

---

## 二、噪声 & 纹波 核心考点

### 1. 基础概念

- **纹波**：电源直流上的低频周期性波动，多由开关电源整流、开关频率导致
- **噪声**：高频随机干扰，包含开关噪声、串扰、辐射干扰、地弹噪声

### 2. 产生原因

1. 开关电源 MOS 管高频开关、二极管反向恢复
2. 高速数字信号跳变，瞬时大电流引发地弹
3. 走线过长、无完整参考地、阻抗不匹配
4. 滤波电容选型 / 摆放不合理

### 3. 抑制措施（多选高频）

- 就近放置**0.1μF 陶瓷去耦电容 + 大容量电解 / 钽电容**组合滤波
- 完整地平面、分割地合理处理
- 增加磁珠、LC 滤波电路
- 降低环路面积，减小干扰耦合
- 开关回路走线尽量短

---

## 三、无源器件：电容 必背

1. 作用：去耦、滤波、储能、隔直、稳压、抑制纹波
2. 高频特性：
    
    - 陶瓷电容：高频特性好，适合**高频去耦**
    - 电解电容：低频大容量，适合**低频纹波抑制**
    
3. 关键常识：
    
    - 电容串联：耐压升高、容量减小
    - 电容并联：容量增大、耐压不变
    - 高频下电容存在寄生电感，会失效

## 五、阻抗匹配（重中之重）

### 1. 目的

避免信号反射、振铃、过冲 / 下冲，保证高速信号完整，降低 EMI 电磁干扰

### 2. 常见场景

- 高速数据线、时钟线、差分信号、射频 / 高速互连
- PCB 微带线、带状线都有固定特征阻抗

### 3. 常用匹配方式

- 串联端接（源端匹配）：最常用，单板硬件高频
- 并联端接、戴维南端接
- 差分线严格控制差分阻抗（典型 100Ω）

### 4. 常见阻抗值

- 单端信号：50Ω/60Ω
- 差分高速：100Ω

## 1. RC 电路的时间常数

- **题目**：RC 电路的时间常数 τ = ()。
    
    A. C/R
    
    B. 1/RC
    
    C. RC
    
    D. R/C
- **答案**：C
- **解析**：RC 电路的时间常数是电阻与电容的乘积，即 τ = RC。它代表电容充电 / 放电至稳态值约 63.2% 所需的时间。
- **考点补充**：时间常数越大，充放电过程越慢；在滤波、延时电路中是核心参数。

## 2. 器件壳温的精准测试

- **题目**：当需要精准测试器件壳温时，需采用 ()
    
    A. 优选红外测温仪
    
    B. 优选热电偶测温仪
    
    C. 热电偶和红外测温仪都可以
- **答案**：B
- **解析**：热电偶为接触式测温，可直接与器件壳接触，受外界干扰小，精度高；红外测温为非接触式，受器件表面发射率、环境光影响，精度低于热电偶。
- **考点补充**：硬件测试中，接触式热电偶是 PCB / 器件温度测试的标准方案。

## 3. 存储介质的容量密度

- **题目**：请选择一下容量密度最大的介质 ()
    
    A. SLC
    
    B. QLC
    
    C. PLC
    
    D. TLC
- **答案**：C
- **解析**：按单单元存储位数排序：SLC（1bit）< TLC（3bit）< QLC（4bit）< PLC（5bit），层数越多 / 单单元存储位数越高，容量密度越大，因此 PLC 最大。
- **考点补充**：PLC 是新一代闪存技术，目前主流 SSD 仍以 TLC/QLC 为主。

## 4. 3.3V TTL 与 CMOS 电路的噪声容限

- **题目**：在 3.3VTTL 和 CMOS 电路设计中，需要保证合格的噪声容限，即 Vohmin-Vihmin 和 Vilmax-Volmax 应大于或等于___
    
    A. 1.0V
    
    B. 0.2V
    
    C. 0V
    
    D. 0.4V
- **答案**：D
- **解析**：3.3V TTL/CMOS 标准中，高电平噪声容限（Vohmin-Vihmin）和低电平噪声容限（Vilmax-Volmax）均要求≥0.4V，确保信号抗干扰能力。
- **考点补充**：噪声容限越大，电路抗干扰能力越强，是数字电路设计的关键指标。

## 5. 处理器存储器访问速率

- **题目**：处理器对存储器的访问，速率最快的是？
    
    A. FLASH
    
    B. Cache
    
    C. DDR
    
    D. Register
- **答案**：D
- **解析**：按访问速率从快到慢排序：Register（寄存器，CPU 内部）> Cache（高速缓存）> DDR（内存）> FLASH（闪存）。寄存器离运算核心最近，访问延迟最低。
- **考点补充**：CPU 存储层次的核心逻辑是 “离核心越近，速度越快、容量越小、成本越高”。

## 6. I2C 总线的总电容容量限制

- **题目**：I2C 总线规范允许总线的电容总容量不能大于多少？（）
    
    A. 300pf
    
    B. （后续选项未显示）
- **常见答案**：标准 I2C 总线规范中，总线电容负载上限为 **400pF**（部分低速模式为 100pF/300pF，笔试中常考 400pF）。
- **解析**：总线电容过大会导致信号上升 / 下降沿变缓，影响时序和信号完整性，因此规范对总线总电容有明确限制。
- **考点补充**：硬件设计中，I2C 设备数量过多、走线过长都会增加总线电容，需提前评估。
