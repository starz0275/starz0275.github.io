# Obsidian 本地笔记库

在 Obsidian 中 **打开本文件夹**（`vault/`）作为仓库根目录，与网站源码分离。

## 目录说明

| 文件夹 | 用途 | 是否同步到网站 |
|--------|------|----------------|
| `面试/准备中/` | 待面试公司的调研、岗位 JD、临时笔记 | 否，面试结束后归档或删除 |
| `面试/总结模板.md` | 单场面试复盘写作模板 | 否 |
| `草稿/` | 未完成的笔记、随手记 | 否 |
| `私人/` | 完全不公开的内容 | 否（已 gitignore） |

## 与网站的关系

```
starz0275.github.io/
├── vault/          ← Obsidian 打开这里（本地工作区）
├── docs/           ← 发布到网站的学习笔记、面试复盘
├── blog/           ← 发布的博客
└── src/pages/      ← 简历等页面
```

**发布流程**：在 `vault/` 里写完 → 整理后复制/移动到 `docs/` 或 `blog/` → `yarn build` 验证 → `yarn deploy`。

### 面试内容怎么放

| 内容 | 放哪里 |
|------|--------|
| 准备中的公司调研、JD 摘抄 | `vault/面试/准备中/` |
| 已完成的面试复盘 | `docs/面试复盘/已面试/` |
| 题库汇总 | `docs/面试复盘/题库/` |
| 进度追踪 | `docs/面试复盘/面试进度总览.md` |
